## 1. Ключи и доступы (вписываются в `.env`)

| # | Что | Переменная в `.env` | Как получить (кратко) | Что включает | Приоритет | Статус |
|---|-----|--------------------|----------------------|--------------|-----------|--------|
| 1 | **Telegram-бот** | `TELEGRAM_BOT_TOKEN` + `TELEGRAM_CHAT_ID` | @BotFather + написать боту | Алерты и дневной дайджест | — | ✅ готово |
| 2 | **goszakup токен** | `GOSZAKUP_TOKEN` | Письмо в АО «ЦЭФ» (support@ecc.kz) **или** кабинет: Профиль участника → Выпуск токена | Основной поток тендеров (GraphQL API) | 🔴 высокий | ☐ |
| 3 | **Живой скрапинг** | `DR_SCRAPE_LIVE=1` | Ключ НЕ нужен, просто впиши `1` | mitwork, med/fms.ecc, mp.kz, RSS-новости | 🔴 высокий | ☐ |
| 4 | **data.egov ключ** | `DATA_EGOV_APIKEY` | egov.kz → Кабинет разработчика → Создать ключ (бесплатно, сразу) | Новые компании по ОКЭД (опережающий) | 🟡 средний | ☐ |
| 5 | **Wordstat токен** | `YANDEX_DIRECT_TOKEN` | Яндекс.Директ (черновая кампания) → OAuth → заявка в поддержку | Латентный спрос (рост запросов) | 🟢 низкий | ⚪ по желанию |
| 6 | **hh.ru токен** | `HH_TOKEN` | dev.hh.ru → Создать приложение (одобрение ~дни) | Вакансии watchlist-сетей | 🟢 низкий | ⚪ по желанию |
| 7 | **SMTP (почта)** | `SMTP_HOST/PORT/USER/PASSWORD/FROM` | Данные почтового ящика (Gmail/Яндекс/свой) | Реальная отправка точечных писем | 🟢 низкий | ⚪ по желанию |

**Как применить любой ключ:** открыл `.env` → вставил значение в нужную строку → сохранил →
запустил `python -m demandradar --once`. Всё. Без ключа коннектор просто работает на
тестовых данных (mock).

## 3. Рекомендуемый порядок действий

1. **Отправь письмо в ЦЭФ** на токен goszakup (текст готов в ONBOARDING_ACCESS.md) — это
   самый долгий по времени пункт, запусти его первым.
2. **Впиши `DR_SCRAPE_LIVE=1`** — мгновенно оживляет 4 источника без всяких ключей.
   Скажи Клоду — он проверит, что парсеры совпали с реальной вёрсткой.
3. **Получи ключ data.egov** — 5 минут, бесплатно, сразу.
4. Wordstat / hh / SMTP — когда дойдут руки, они не критичны.
5. Дождись правок от шефа по ТРУ/ОКЭД/watchlist → отдай Клоду.
