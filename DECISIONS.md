# DECISIONS — журнал технических решений

Формат: D-N | решение | альтернативы | почему.

- **D-1** | Синхронное ядро (httpx sync), без asyncio | asyncio, aiohttp | десятки источников с опросом раз в минуты — не тот масштаб, где async окупает сложность; надёжность 24/7 и простота отладки важнее. Параллелизм позже: поток на коннектор (Э7), интерфейс Connector не изменится.
- **D-2** | Собственный ретрай/backoff в Fetcher (~40 строк) | tenacity, backoff | меньше зависимостей; нужна тонкая связка с троттлингом по хосту, Retry-After и ProxyProvider.
- **D-3** | pydantic v2 для DemandSignal и валидации | dataclasses, attrs | бесплатная сериализация/валидация на границе коннектор→ядро, где данные грязные.
- **D-4** | SQLite через stdlib sqlite3 + repo-слой на чистом SQL | SQLAlchemy | ORM не окупается на 3-5 таблицах; repo-слой изолирует SQL для будущего Postgres.
- **D-5** | CLI на argparse | click, typer | ноль зависимостей, флагов немного.
- **D-6** | Дашборд: FastAPI + Jinja2 (серверный HTML) (Э3) | SPA (React/Vue), Streamlit | по брифу «лёгкий и бесплатный»; серверный HTML проще деплоить и он же отдаст health-check и API отчёта.
- **D-7** | Конфиг: секреты в .env, реестр источников и категории в config/*.yaml | всё в .env / всё в коде | YAML-реестр = «добавить источник без правки кода» и документация одновременно.
- **D-8** | goszakup: основной протокол GraphQL, инкапсулированный клиент с фолбэком на REST v3 и аварийным HTML-поиском | только GraphQL | 3 независимых пути к тем же данным = устойчивость 24/7; проверено, что HTML-поиск публичен.
- **D-9** | Ключ дедупа: sha256(source + source_id), где source_id — стабильный ID источника (номер объявления+лота) | хэш заголовка/нечёткий матчинг | детерминированно и дёшево; кросс-источниковый нечёткий дедуп (один тендер на goszakup и агрегаторе) — осознанно НЕ на старте, добавим при реальных дублях (Э4, по trdBuyNumberAnno).
- **D-10** | ИИ-контракт: провайдер возвращает None при любой ошибке, никаких исключений наружу; фабрика по DR_LLM_PROVIDER, дефолт null | исключения, обязательный LLM | ядро обязано работать без ИИ (бриф); деградация тестируется.
- **D-11** | Ollama-рекомендация qwen3:8b, температура 0.1 | llama3.x | Qwen сильнее в русском при том же размере; модель — строка конфига.
- **D-12** | Отдельный git-репозиторий в Desktop/demandradar (в домашней папке обнаружен случайный git-репо; наш — вложенный независимый) | чистить домашний репо | не трогаем чужое окружение.
- **D-13** | ConnectorMode.MOCK/LIVE вычисляется по наличию env-ключей коннектора (required_env), вручную можно форсировать | флаг в конфиге | ровно как в брифе: переключение без правки кода.
- **D-14** | Курсоры и health коннекторов — таблица connector_state в SQLite | json-файлы | транзакционность с записью сигналов + данные для мониторинга в дашборде.
